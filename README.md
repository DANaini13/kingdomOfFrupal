# kingdomOfFrupal

### Environment Requirements:
> 1. JDK 1.8 +
> 2. Maven

### Running Instructions:
Server:
> 1. Transfer to the ```FrupalServer``` drictory
> 2. Compile the program with ```mvn package```
> 3. Rum the program with ```mvn exec:java```
Client
> 1. Transfer to the ```FrupalClient``` drictory
> 2. Compile the program with ```mvn package```
> 3. Rum the program with ```mvn exec:java```
